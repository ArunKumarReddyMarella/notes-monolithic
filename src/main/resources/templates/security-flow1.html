Notes Management Flow

Creating a Note
1
User submits note creation form with title, description, and optional file attachment
2
NotesController receives request with @PreAuthorize("hasAnyRole('ADMIN', 'USER')")
3
NotesService.saveNotes() processes note data and optional file
4
NotesRepository.save() persists note to database
5
Success/failure response returned to user
Retrieving Notes
1
Client requests notes list with pagination parameters
2
NotesController processes request with authentication check
3
NotesService.getUserNotes() fetches user-specific notes with pagination
4
NotesRepository.findByCreatedByAndIsDeletedFalse() retrieves database records
5
Records mapped to DTOs and returned in NotesResponse with pagination metadata
Favorite Notes Handling
1
User marks note as favorite
2
NotesService.favoriteNotes() creates FavouriteNote entity
3
FavouriteNoteRepository.save() persists record
4
User retrieves favorite notes list
Recycle Bin Flow
1
User soft-deletes a note (marks isDeleted=true)
2
User views recycle bin via getUserRecycleBinNotes()
3
NotesRepository.findByCreatedByAndIsDeletedTrue() retrieves deleted notes
4
User can restore or permanently delete notes

Security Configuration

@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
  http
    .csrf(AbstractHttpConfigurer::disable)
    .authorizeHttpRequests(auth -> auth
      .requestMatchers("/api/v1/auth/**").permitAll()
      .anyRequest().authenticated()
    )
    .httpBasic(Customizer.withDefaults())
    .sessionManagement(session -> session
      .sessionCreationPolicy(SessionCreationPolicy.STATELESS)
    )
    .addFilterBefore(jwtFilter, UsernamePasswordAuthenticationFilter.class);
  return http.build();
}
